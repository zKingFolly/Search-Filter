# Search Filter

A simple vanilla JavaScript project that implements a live search filter for a list of items. This project demonstrates how to filter content on the page as the user types, without any frameworks.

## Features

- Live filtering of list items as the user types
- Case-insensitive matching
- Simple and lightweight vanilla JavaScript implementation
- Clean and easy-to-read code

## Usage

1. Open the HTML file in your browser.
2. Type a query in the search box.
3. The list updates instantly to show matching results.

## Project structure

- `index.html` - Contains the page layout and search input.
- `styles.css` - Styles the page and list elements.
- `script.js` - Contains the filtering logic.
